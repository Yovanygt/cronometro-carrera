# Cronómetro de Carrera Web

Este proyecto consiste en una aplicación web desarrollada con **HTML, CSS y JavaScript**, la cual permite registrar el tiempo de llegada de diferentes competidores en una carrera.

El sistema funciona como un cronómetro que inicia cuando comienza la carrera y permite registrar el momento en que cada participante llega a la meta. A partir del primer competidor registrado, el sistema calcula automáticamente la diferencia de tiempo con los demás participantes.

## Funcionalidades

- Iniciar cronómetro de la carrera
- Registrar llegada de competidores
- Mostrar el tiempo de cada competidor
- Calcular la diferencia con respecto al primer lugar
- Reiniciar la carrera

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Uso de la aplicación

1. Presionar **Iniciar Carrera** para comenzar el cronómetro.
2. Cada vez que un competidor llegue a la meta, presionar **Registrar Llegada**.
3. El sistema mostrará los tiempos registrados y la diferencia respecto al primer competidor.
4. Presionar **Reiniciar** para comenzar una nueva carrera.
